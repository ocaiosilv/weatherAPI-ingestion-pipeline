# weatherAPI-ingestion-pipeline

A simple weather data project built with Python, SQL, and APIs.  
The project focuses on collecting, processing, and storing weather information for analysis and automation purposes.

## Technologies
- Python
- Pandas
- SQL
- APIs
- SQLAlchemy

## About
This project is being developed to practice data engineering concepts such as data pipelines, API integration, data transformation, and database management.

## Instructions
To run the pipeline properly, make sure that PostgreSQL is installed and configured locally.

Since the `.env` file is not included in the repository, create a local `.env` file with the required environment variables before running the project. Then, execute `create_db.py` before running `pipeline.py`.

Also, make sure that all dependencies listed in `requirements.txt` are installed correctly.
